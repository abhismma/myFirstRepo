Client & Server Communication

The client is your browser or app, and the server is a computer that responds with data. When you type a URL and hit Enter, a request is sent to a server.

The Journey of a Web Request

  1. User enters a URL in the browser.
  2. Browser sends a DNS request to find server IP.
  3. HTTP request is sent to the server.
  4. Server processes the request.
  5. Response is sent back and rendered on the browser.
